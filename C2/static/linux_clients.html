Issue Command Help

Advanced Commands Available in the Client Manage Page

Client Commands

Kill Client: Sends a command to tell the client to kill itself (Turn Off/Close)

Uninstall Client: Will send a command to tell the Client to Uninstall itself from the Host.

Update Client: Sends a command and link to file for the Client to run and then Uninstall itself.

 

Simple Commands

Open URL: Sends a command to client to open a URL with Options for doing it Hidden.

Start Process: Sends command to client to start a process on the host.

Download and Run: Sends command to client to download and run a file. Options for Adding to Startup.

 

Power Commands

Shutdown: Sends command to shutdown the Host.

Restart: Sends command to resart the Host.

DDoS Control Help

 

Attack Methods

TCP Flood: The most common attack involves sending numerous SYN packets to the victim.

UDP Flood: Sends UDP packets, typically large ones, to single destination or to random ports. 

Slowloris: Sends HTTP headers to the target site in tiny chunks as slow as possible.

HULK v3: Used to attack web servers by generating unique and obfuscated traffic volumes.

GoldenEye: A Layer 7 attack using KeepAlive and NoCashe.

HTTPS Flood: Send seemingly legitimate set of HTTPS GET or POST requests.

CCTV Attack: Remotely disable CCTV camera streams, like in spy movies.

Attack Settings

IP Address and Port: The IP and the Port you wish to attack.

Attack Time: How long the Attack will last. You may lose connection to the Client while attacking.
